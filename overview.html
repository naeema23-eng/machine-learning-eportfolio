Module Overview

This e-portfolio presents my learning and development throughout the Machine Learning module. The module provided both theoretical and practical understanding of machine learning techniques, moving from foundational concepts such as exploratory data analysis, regression, and clustering, to neural networks, deep learning, and model evaluation.

Module Learning Focus

A central theme of this module was the application of machine learning methods to real-world problems while critically considering technical limitations, data quality, uncertainty, and professional responsibility. Across the module, I engaged with both classical machine learning approaches and neural network models, developing a stronger understanding of when different methods are appropriate and what risks may affect performance.

Key Areas of Development

  • understanding the role of data preparation, exploratory analysis, and feature selection
  • applying regression and clustering methods to structured datasets
  • developing knowledge of artificial neural networks and deep learning workflows
  • evaluating model performance using appropriate metrics
  • strengthening critical awareness of bias, overfitting, interpretability, and system reliability
  • developing collaboration, coordination, and leadership skills through project-based work

How This Portfolio Is Organised

The portfolio is structured to show development across the full module. It includes unit-by-unit evidence, my contribution to the Unit 6 development team project, my Unit 11 individual project on object recognition, an evaluation comparing the final project against the earlier project stage, collaboration evidence, and a final reflective section supported by references.

Unit-by-Unit Evidence of Learning

This section brings together evidence of my development across the Machine Learning module. It shows how my understanding progressed from foundational data analysis and regression methods to clustering, neural networks, model evaluation, deep learning, and future-facing machine learning topics.

Units 1–3: Foundations, EDA, Regression Thinking and System Risk

In the opening part of the module, I developed a foundational understanding of machine learning as both a technical and socio-technical discipline. My work focused on exploratory data analysis using the Auto-MPG dataset, including missing value handling, skewness and kurtosis analysis, correlation analysis, and regression thinking. This stage highlighted the importance of data quality, multicollinearity, and the limitation that correlation does not imply causation.

My reflection also connected these technical concepts to wider issues such as Industry 4.0, Industry 5.0, system vulnerability, and the need for robust and human-centred machine learning in real-world environments.

Evidence to insert here: EDA screenshots, correlation heatmap, scatter plot, early reflection excerpt.

Units 4–6: Regression, Similarity, Clustering and Applied Machine Learning

In these units, I expanded my understanding of classical machine learning through practical work in linear regression, similarity measurement, and clustering. I analysed the relationship between population and GDP using linear regression, calculated Jaccard coefficients for binary similarity comparison, and explored clustering conceptually before applying K-Means using Python.

These activities strengthened my ability to interpret model performance critically, especially in relation to feature selection, scaling, cluster quality, and the limitations of simple models when dealing with complex real-world data.

Evidence to insert here: regression output, Jaccard work, clustering notes, elbow plot, silhouette score.

Unit 6: Development Team Project Context

Unit 6 also marked an important transition from individual technical exercises to collaborative project work. Through the Airbnb NYC 2019 analysis, I contributed to a real-world business-focused machine learning task involving pricing prediction and market segmentation. This project developed both my technical skills and my ability to work within a virtual development team environment.

A dedicated page later in this portfolio presents the project in detail, including my individual contribution, team role, methodology, and reflection.

Evidence to insert here: one screenshot linking this unit section to the full Unit 6 project page.

Unit 7: Introduction to Artificial Neural Networks

In Unit 7, I explored the foundations of artificial neural networks through perceptron-based activities. I first examined how a simple perceptron produces outputs from weighted inputs and a threshold function. I then implemented a perceptron for the AND operator, showing how weights can be updated through training. Finally, I developed a multilayer perceptron with sigmoid activation to solve the XOR problem, demonstrating why hidden layers and non-linear activation are necessary for more complex learning tasks.

This unit was important because it formed the conceptual bridge between classical machine learning and the more advanced deep learning work completed later in the module.

Evidence to insert here: perceptron output, AND notebook result, XOR prediction screenshot, training error graph.

Units 8–10: Optimisation, CNNs and Critical Deep Learning Awareness

In the later middle stage of the module, my understanding moved further into neural network optimisation and deep learning. I engaged with gradient descent and cost function behaviour, which strengthened my understanding of how models learn through iterative error reduction. This supported later work on convolutional neural networks and deep learning practice.

At the same time, my learning became more critical. I increasingly considered overfitting, evaluation depth, bias, interpretability, and ethical implications, particularly in image classification and other higher-stakes machine learning applications.

Evidence to insert here: gradient descent graph, CNN notes, model training screenshots, critical reflection excerpts.

Unit 11: Evaluation Metrics and Individual Deep Learning Project

Unit 11 brought together the technical foundations developed throughout the module in a more advanced independent project. I worked on object recognition using CIFAR-10 and compared a custom CNN with a MobileNetV2 transfer learning approach. This required dataset preparation, augmentation, validation design, model training, and critical comparison of results using multiple evaluation methods.

This unit significantly strengthened my independent problem-solving ability and my understanding of how deep learning performance depends not only on model complexity, but also on data quality, class similarity, and careful evaluation.

Evidence to insert here: validation curves, confusion matrix, architecture screenshot, results table.

Unit 12: Future of Machine Learning

The final stage of the module extended my perspective beyond current methods to future machine learning developments. Key ideas included self-supervised learning, neural architecture search, and edge AI, along with wider ethical concerns related to fairness, privacy, sustainability, and misuse.

This helped consolidate one of the most important lessons of the module: machine learning must be understood not only as a set of algorithms, but as a field shaped by real-world constraints, governance, and human impact.

Overview

This section presents the practical machine learning activities completed throughout the module. These activities demonstrate progression from exploratory data analysis and regression, to clustering, neural networks, deep learning, and model evaluation.

Unit 2 & 3: Exploratory Data Analysis and Regression

I conducted exploratory data analysis (EDA) on the Auto MPG dataset, including data cleaning, handling missing values, and analysing statistical measures such as skewness and kurtosis.

A strong negative correlation between vehicle weight and MPG was identified and visualised using scatter plots and correlation heatmaps. This helped transform statistical relationships into interpretable insights.

A simple linear regression model was implemented to quantify this relationship. While the model captured the general trend, it demonstrated the limitation of using a single predictor, highlighting the importance of feature selection.

Unit 4: Linear Regression Analysis

This activity extended regression analysis using population and GDP datasets. Data merging, statistical analysis, and correlation measurement were performed before applying regression.

The results showed a positive correlation between population and GDP, but also revealed that the model explained only part of the variation. This demonstrated that correlation does not guarantee predictive accuracy.

This activity strengthened my ability to critically interpret model outputs rather than relying only on numerical results.

Unit 6: Clustering and Unsupervised Learning

Using the Iris dataset, I applied K-Means clustering to identify patterns without labelled data. Data standardisation was performed to ensure fair feature contribution.

The elbow method was used to determine the optimal number of clusters, and the silhouette score was used to evaluate clustering performance.

This activity highlighted that unsupervised learning requires interpretation and analytical judgement rather than purely algorithmic output.

Unit 7: Neural Networks – Perceptrons

This unit introduced neural networks through perceptron models, demonstrating how weighted inputs and activation functions produce outputs.

The perceptron was used to model an AND logic gate, illustrating supervised learning through weight updates. A multi-layer perceptron was then implemented to solve non-linear problems, showing the importance of hidden layers.

This activity provided a strong conceptual foundation for understanding deep learning.

Unit 8: Gradient Descent and Optimisation

This activity explored gradient descent as an optimisation algorithm used to minimise cost functions and improve model performance.

The iterative updates of model parameters demonstrated how cost decreases over time, and how learning rate affects convergence behaviour.

This provided practical insight into how machine learning models are trained and optimised.

Unit 9: Convolutional Neural Networks (CNN)

This activity introduced deep learning using convolutional neural networks for image classification with the CIFAR-10 dataset.

The workflow included dataset loading, image visualisation, preprocessing, and preparation of training and validation sets.

This demonstrated how CNNs automatically extract spatial features, making them highly effective for computer vision tasks.

Unit 11: Model Performance Evaluation

This activity focused on evaluating models using confusion matrices and performance metrics such as accuracy, precision, recall, and F1-score.

It highlighted that accuracy alone is insufficient, especially in imbalanced datasets, and that multiple metrics are required to fully understand model performance.

This strengthened my ability to critically assess machine learning models in real-world contexts.

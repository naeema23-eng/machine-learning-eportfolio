Welcome

This e-portfolio presents my learning and development throughout the Machine Learning module. It brings together discussion work, practical activities, seminar-linked tasks, project contributions, and critical reflections to demonstrate both technical progress and professional growth.

It reflects how my understanding developed across both collaborative and independent work, from foundational machine learning concepts to the design, evaluation, and critical interpretation of more advanced models in realistic contexts.

Purpose of This Portfolio

The purpose of this portfolio is to provide a structured and evidence-based account of my work across the module. It shows how my understanding developed from foundational machine learning concepts to more advanced topics such as neural networks, deep learning, model evaluation, and real-world machine learning risk.

It also demonstrates the integration of technical knowledge with wider academic and professional skills, including critical reading, collaborative discussion, reflective writing, and responsible thinking about the limitations and implications of machine learning systems.

Portfolio Structure

  • Discussions: collaborative discussion posts, peer responses, and summary reflections demonstrating critical engagement with academic literature
  • Activities: practical unit work including coding exercises, data analysis, and applied machine learning tasks
  • Seminars: seminar preparation, outputs, and key learning points linked to weekly topics
  • Projects: major assessed work including the Unit 6 team project and Unit 11 individual deep learning project
  • Reflections: structured reflection on learning development, including the final 1,000-word reflective review
  • References: complete Harvard-style reference list used across the portfolio
  • About: personal background and professional interest in machine learning within healthcare

Module Development Themes

Across this module, I developed knowledge in exploratory data analysis, regression, clustering, artificial neural networks, optimisation, deep learning, and model evaluation. These technical areas were supported by an increasing focus on how machine learning systems operate in real-world environments, particularly in terms of reliability, interpretability, and risk.

A key theme throughout this portfolio is the transition from viewing machine learning as purely a technical discipline toward understanding it as a socio-technical system. This includes recognising the importance of data quality, system design, human oversight, and ethical responsibility in ensuring that models are not only accurate, but also safe and appropriate for practical use.

This perspective is particularly relevant to my professional context in healthcare, where machine learning systems must operate under uncertainty and where errors can have direct real-world consequences. As a result, this portfolio emphasises not only model performance, but also robustness, transparency, and responsible deployment.

Submission Overview

This e-portfolio represents the final submission for the Machine Learning module. It provides a comprehensive record of my academic and practical development, supported by evidence from completed tasks, project outputs, and reflective analysis.

The portfolio is designed to demonstrate not only technical competence in machine learning methods, but also the ability to critically evaluate models, engage with academic research, and apply learning within a professional and ethical context.
